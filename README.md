# Currency API

یک API ساده Node.js برای دریافت نرخ ارز از سایت sarafi.af و ارائه endpoint به صورت JSON.

## راه‌اندازی محلی

1. نصب وابستگی‌ها:

```
npm install
```

2. اجرای سرور:

```
npm start
```

سرور روی http://localhost:3000/rates اجرا می‌شود.

## انتشار روی Render

راهنمای کامل را در پیام قبلی مشاهده کنید.
